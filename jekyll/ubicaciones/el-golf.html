Ubicado entre Isidora Goyenechea y Av. El Bosque Norte, Edge Cowork El Golf tiene tres pisos con fachada hacia San Sebastián. Puedes llegar caminando desde el metro El Golf o metro Tobalaba, con una ubicación perfecta para la persona que le gusta estar en un distrito de negocios. Este es el primer local de Edge Cowork, con una decoración rústica e industrial que te permite estar en un entorno serio de trabajo pero a la vez relajado. El espacio está al lado de restaurantes, bancos, y matrices de empresa grandes.

El cowork tiene tres sectores para coworkers. En el primer piso hay un coffee bar para los miembros y recepción para visitas. El segundo piso incluye una variedad de escritorios para tanto trabajar sentado como parado. Tenemos una gran mesa redonda para 8 a 10 personas que pueden conectarse a trabajar tranquilamente. Este piso incluye un lounge cómodo para reuniones informales, como también para trabajar en otro entorno. Este piso incluye Lockers para guardar tus cosas, baños y una cabina privada para esas reuniones 1 a 1 o si necesitas tomar una llamada importante. El tercer piso incluye una variedad de oficinas privadas y un espacio privado con Dedicated Desks para los que quieren un escritorio fijo. Este piso además tiene una sala de reuniones para 6 personas, kitchenette y baños.

Conoce Edge en El Golf

¿Qué tiene Edge en el Golf?

Espacio
Espacio para hacer meetups, talleres y charlas pequeñas para 20 personas, después de las 7:30pm. Muebles cómodos con enchufes para computadores y teléfonos. Áreas comunes tipo lounge para trabajar, recibir visitas o descansar.
Reuniones
Una sala de conferencias para 6 personas. Dos cabinas privadas para llamar por teléfono, conferencias o reuniones uno a uno, con uso ilimitado.
Conectividad
Internet estable y de alta velocidad con enlace dedicado 1:1 de 100MB.n.
Servicios
Café de grano, té y agua purificada. Community manager in-situ disponible para atenderte. Servicio de limpieza diario. Impresoras a color y scanner.
Comodidades
Control de Acceso Smart para entrar y salir 24/7. Lockers y estacionamiento de bicicletas. Estacionamientos para autos sujeto a disponibilidad.he time and space for you to host your own.
